PROJECTS

Blocipedia

Blocipedia is a Ruby on Rails application that allows users to create wikis and collaborate on other users' wikis. Users can pay to upgrade their membership, allowing them to view and create private wikis.

Bloc Chat

Bloc Chat uses Firebase and AngularJS to create a chat application. Users can create their own chatrooms and send messages to each in real-time.

Kele Ruby Gem API

The Kele gem is an API client to access the Bloc.io's student data. Rather than using cURL to access Bloc's API, you can use this gem to access student information in your own Rails application.

Bloc Jams

Bloc Jams uses HTML/CSS, JavaScript, and Angular to create a music library and streaming player with a responsive design. The app will display music albums, track listings, and provide the ability to play tracks.
